# Визуализатор графа зависимостей Cargo

## 1. Общее описание

Инструмент командной строки для построения и визуализации графа зависимостей пакетов из экосистемы Rust (Cargo).

*Текущая версия реализует базовый прототип с парсингом и валидацией конфигурационных параметров, передаваемых через командную строку.*

---

## 2. Описание функций и настроек

Программа настраивается через следующие опции командной строки:

-   `--package <ИМЯ>`: **(Обязательный)** Имя анализируемого пакета.
-   `--repo-url <URL>`: **(Обязательный)** URL-адрес репозитория пакета.
-   `--test-mode`: Флаг, включающий режим работы с тестовым репозиторием.
-   `--ascii-tree`: Флаг, который на будущих этапах будет включать вывод графа в виде ASCII-дерева.
-   `--filter <ПОДСТРОКА>`: Задает подстроку для фильтрации пакетов.
-   `-h`, `--help`: Показывает справочное сообщение со всеми доступными параметрами.

---

## 3. Команды для сборки и запуска

Проект написан на Python и не требует предварительной сборки. Для запуска используется интерпретатор `python3`.

### Основная команда для запуска:
```
python3 main.py [ПАРАМЕТРЫ]
```

На данном этапе тесты не предусмотрены.

---

## 4. Примеры использования

### Успешный запуск

Команда:
```
python3 main.py --package serde --repo-url https://github.com/serde-rs/serde
```

Результат:
```
============================================================
Конфигурация визуализатора зависимостей Cargo
============================================================
Имя пакета:           serde
URL репозитория:      https://github.com/serde-rs/serde
Тестовый режим:       Нет
Режим ASCII-дерева:   Нет
Фильтр пакетов:       (не задан)
============================================================
```

### Демонстрация обработки ошибок

#### Отсутствие обязательных параметров:

Команда:
```
python3 main.py
```

Результат:
```
usage: main.py [-h] --package PACKAGE --repo-url REPO_URL [--test-mode]
               [--ascii-tree] [--filter FILTER]
main.py: error: the following arguments are required: --package, --repo-url
```

#### Передача пустого значения:

Команда:
```
python3 main.py --package "" --repo-url https://github.com/serde-rs/serde
```

Результат:
```
Ошибки валидации:
  - Имя пакета не может быть пустым
```
